monkeBLOCK

  • Redirect blocking Determines if monkeBLOCK will block redirects. It is recommended to keep this ENABLED at all times.
  • Remove malicious elements When ENABLED, monkeBLOCK will remove potentially malicious elements from the webpage that may cause redirects. Could break some webpages. Recommended to keep ENABLED on suspicious websites.
  • StrictMode StrictMode is a mode that will block any and all redirects from one website to another, regardless of website certificate status. Recommended to enable only when browsing deeper parts of the internet.
Issues? Reload monkeBLOCK.
